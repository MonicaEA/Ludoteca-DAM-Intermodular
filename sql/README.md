## Contenido de la carpeta SQL (DOCUMENTACIÓN BBDD)

Para que el sistema funcione, los scripts deben ejecutarse en el siguiente orden:

### 1. [Estructura de la Base de Datos](https://github.com/MonicaEA/CompraVenta-DAM-Intermodular/blob/114d83efc740525b6d4eaf93cf48ffdf250ead4f/sql/creacionTablasSQL.sql)
* **Archivo:** `creacionTablasSQL.sql`
* **Descripción:** Contiene todas las sentencias DDL para la creación de la base de datos `retrojuegos`. 


### 2. [Carga de Datos de Ejemplo](https://github.com/MonicaEA/CompraVenta-DAM-Intermodular/blob/114d83efc740525b6d4eaf93cf48ffdf250ead4f/sql/insercionDatosSQL.sql)
* **Archivo:** `insercionDatosSQL.sql`
* **Descripción:** Script con sentencias DML para poblar las tablas con datos reales de prueba. 
* **Objetivo:** Sirve para que el sistema no arranque vacío y se puedan probar las funciones de búsqueda y filtrado de stock desde el primer momento.

### 3. [Repositorio de Consultas](https://github.com/MonicaEA/CompraVenta-DAM-Intermodular/blob/114d83efc740525b6d4eaf93cf48ffdf250ead4f/sql/consultasSQL.sql)
* **Archivo:** `consultasSQL.sql`
* **Descripción:** querys añadiendo algunos ejemplos del temario que hemos dado en clase (básicas, joins, agregaciones y subconsultas).
